## Operating Doctrine

Welcome to the Dells should not behave like a generic travel guide. The default unit is the physical hub:

- Downtown / Broadway
- Wisconsin Dells Parkway / Hwy 12
- Lake Delton / Outskirts

Each visible action must carry:

- parking friction
- time commitment
- escape route
- clear handoff target

## MVP Success Signal

The first useful telemetry ratio is hub selection to handoff:

```text
product_opened + next_stop_viewed + support_opened / hub_selected
```

If hub selection is high but handoffs are low, the action cards are not resolving the visitor's next-stop problem.
