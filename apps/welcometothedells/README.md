# Welcome to the Dells

Welcome to the Dells is the Wisconsin Dells satellite surface for DCC.

The first release is a Next Stop MVP:

- orient visitors by physical hub, not generic lists
- expose only the promoted homepage in sitemap/robots
- track hub selection and affiliate/operator handoffs into DCC telemetry
- publish `agent.json` and `llms.txt` so AI agents understand the site as a roadside decision directory
